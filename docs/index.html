Features

Everything You Need

Built for speed, security, and simplicity. Everything you need to manage your proxy panel.

High Performance

256KB relay buffers, TCP_NODELAY for low latency, and fully async I/O for maximum throughput on Hugging Face Spaces.

Anti-Fingerprint

Pre-handshake WebSocket validation, plain-text root endpoint, and proper TLS configuration to avoid detection.

Modern Dashboard

Dark/Light theme with purple accent, bilingual (EN/FA), responsive design, real-time charts, and smooth animations.

Subscription System

Beautiful subscription pages with usage progress bars, expiry countdown, and standard subscription-userinfo header for all clients.

Quota Management

Per-inbound traffic limits, connection caps, and expiry dates. Full control over every configuration you create.

Get Your Hugging Face Token

Paste your token above and click "Deploy". The script creates a new Space and uploads all files automatically.
